介绍

Node.js是一个Javascript运行环境(runtime)。实际上它是对Google V8引擎进行了封装。V8引 擎执行Javascript的速度非常快，性能非常好。Node.js对一些特殊用例进行了优化，提供了替代的API，使得V8在非浏览器环境下运行得更好。 --来着百度百科

优点

  1. 高并发（最重要的优点）
  2. 适合I/O密集型应用
  3. 更适应敏捷开发
  4. 开发速度更快,维护成本低
  5. 社区活跃
  6. 丰富的包(https://www.npmjs.com/)

缺点

  1. 不适合CPU密集型应用
  2. 只支持单核CPU，不能充分利用CPU
  3. 可靠性低，一旦代码某个环节崩溃，整个系统都崩溃
  4. 开源组件库质量参差不齐，更新快，向下不兼容
  5. Debug不方便，错误没有stack trace
